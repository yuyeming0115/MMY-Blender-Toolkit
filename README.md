# MMY Blender Toolkit

MMY 系列 Blender 效率优化工具集，适用于 Blender 4.5+。

## 功能

### 快速后缀保存

在「文件 → 另存为」对话框中，自动显示后缀按钮，一键添加常用后缀：

```
[快速后缀:] [_Mesh] [_Mat] [_Rig] [⚙] [▼]
```

- 点击后缀按钮 → 自动修改文件名（如 `test.blend` → `test_Mesh.blend`）
- 点击 ⚙ → 打开插件偏好设置，编辑后缀列表
- 点击 ▼ → 切换预设（完整流程/动画流程/渲染流程等）

### UV孤岛缝合边

在编辑模式下，一键沿UV孤岛边界自动标记缝合边，无需切换到UV编辑界面。

**入口**：
- N面板：3D视图 → N面板 → "MMY工具" 标签页
- 右键菜单：编辑模式下右键 → 底部操作项
- 搜索菜单：F3 → 搜索 "UV孤岛"

**特点**：
- 保留已有缝合边，只添加新检测到的边界
- 支持自定义快捷键（在偏好设置中配置）

### 导入FBX

在对象模式下，快速导入FBX模型文件。

**入口**：
- N面板：3D视图 → N面板 → "MMY工具" 标签页 → 导入工具

**特点**：
- 点击按钮打开文件选择器
- 动画开关：勾选后导入动画数据，默认关闭

### 预设系统

- 预设保存为 JSON 文件，可迁移到其他电脑
- 默认预设：完整流程、动画流程、渲染流程
- 支持自定义预设，保存/加载/删除

### 偏好设置

在「编辑 → 偏好设置 → 插件 → MMY Toolkit」中：

- 快捷键配置：自定义UV孤岛缝合边操作快捷键
- 左列：编辑当前后缀列表（添加/删除/修改）
- 右列：预设管理（加载/删除）

## 安装

1. 下载 `mmy_toolkit` 文件夹
2. 在 Blender 中：「编辑 → 偏好设置 → 插件 → 安装」
3. 选择下载的文件夹或 zip 包
4. 启用插件

## 配置迁移

预设文件位于：`mmy_toolkit/presets/suffix_presets.json`

复制此文件到其他电脑的相同位置，即可迁移自定义配置。
