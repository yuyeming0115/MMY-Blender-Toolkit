# 项目快捷访问功能开发记录

## 功能概述

整合到顶栏「项目」菜单，读取 Blender 自带的配置文件，提供：
1. **当前文件信息**：显示文件名、复制路径
2. **同目录文件切换**：快速切换同目录其他 .blend 文件
3. **书签功能**：读取 Blender `bookmarks.txt`，点击弹出 File Browser
4. **最近文件**：读取 Blender `recent-files.txt` 的历史文件

---

## 问题记录与解决方案

### 问题 1：书签和最近文件菜单 UI 太宽

**原因**：使用了 `row(align=True)` 同时放置 label 和 operator

**解决**：简化布局，直接用 operator 显示名称作为按钮

### 问题 2：File Browser 目录切换失败

**原因**：`params.directory` 需要特殊处理，且当前空间可能不是 FILE_BROWSER

**解决**：使用 `ImportHelper` 创建专门的文件选择 Operator，弹出标准 File Browser 对话框

### 问题 3：`context.space_data.params` 属性不存在

**报错**：
```
AttributeError: 'Space' object has no attribute 'params'
```

**原因**：当前空间不是 FILE_BROWSER 类型，没有 params 属性

**解决**：不直接设置 params，而是通过设置 `self.filepath` 属性，让 ImportHelper 自动处理

---

## 最终实现方案

### UI 结构

顶栏按钮布局：`项目 | 书签 | 最近文件`

| 按钮 | 功能 |
|------|------|
| 项目 | 当前文件信息 + 同目录文件切换 |
| 书签 | 弹出菜单 → 点击目录弹出 File Browser |
| 最近文件 | 弹出菜单 → 点击直接打开文件 |

### 书签交互

点击书签目录 → 弹出 File Browser 对话框（预设路径）→ 选择文件 → 保存当前 → 打开选中文件

### 核心代码

```python
class MMY_OT_BrowseBookmarkDirectory(bpy.types.Operator, ImportHelper):
    """点击书签弹出 File Browser"""
    bl_idname = "mmy.browse_bookmark_directory"

    directory: bpy.props.StringProperty(subtype='DIR_PATH')

    def invoke(self, context, event):
        if self.directory and os.path.exists(self.directory):
            self.filepath = self.directory  # 预设路径
        return ImportHelper.invoke(self, context, event)
```

| `bookmarks.txt` | `{config}/bookmarks.txt` | `[Bookmarks]` + `[Recent]` |

配置目录定位：从 `bpy.app.binary_path` 推断 portable 版本的 config 目录
