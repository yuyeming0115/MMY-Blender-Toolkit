# 项目快捷访问功能开发记录

## 功能概述

整合到顶栏「项目」菜单，读取 Blender 自带的配置文件，提供：
1. **当前文件信息**：显示文件名、复制路径
2. **同目录文件切换**：快速切换同目录其他 .blend 文件
3. **书签功能**：读取 Blender `bookmarks.txt` 的收藏目录
4. **最近文件**：读取 Blender `recent-files.txt` 的历史文件

## 技术实现

### 数据来源

复用 Blender 原生配置文件，无需自己维护：

| 文件 | 路径 | 内容 |
|------|------|------|
| `bookmarks.txt` | `config/bookmarks.txt` | `[Bookmarks]` 收藏目录 + `[Recent]` 最近目录 |
| `recent-files.txt` | `config/recent-files.txt` | 最近打开的 .blend 文件列表 |

### UI 结构

顶栏「项目」菜单（MMY_MT_ProjectFiles）：

```
┌─ 项目 ─────────────────────────────┐
│ 当前: Model_Mesh.blend    [复制]   │
│────────────────────────────────────│
│ 同目录文件:                        │
│   ✓ Model_Mesh.blend               │
│   Model_Mat.blend                  │
│   Model_Rig.blend                  │
│────────────────────────────────────│
│ 打开目录                           │
│────────────────────────────────────│
│ 书签                               │  ← MMY_MT_BlenderBookmarks
│ 最近文件                           │  ← MMY_MT_BlenderRecentFiles
└────────────────────────────────────┘
```

## 操作符列表

| bl_idname | 功能 |
|-----------|------|
| `mmy.open_project_file` | 打开项目文件（自动保存当前） |
| `mmy.open_project_directory` | 打开项目目录 |
| `mmy.copy_project_path` | 复制路径到剪贴板 |

## 优势

- **复用原生数据**：用户在 File Browser 添加的书签自动同步
- **无需维护**：Blender 自己维护书签和历史文件
- **无冗余代码**：不需要 handlers、JSON 持久化
