# 项目快捷访问功能开发记录

## 功能概述

增强 Blender 项目文件夹的快速访问能力，提供以下功能：

1. **文件浏览器 Header 按钮**：一键打开当前文件所在目录
2. **N 面板项目快捷访问区块**：显示项目信息、书签管理、最近路径
3. **项目书签功能**：收藏常用项目路径，快速切换
4. **自动记忆**：自动记录最近打开的项目路径

## 技术实现

### 1. 数据持久化（config.py）

新增 `presets/project_presets.json` 存储项目书签和最近路径：

```json
{
  "recent_project_paths": ["D:/Project/file1.blend", ...],
  "favorite_project_paths": [{"path": "...", "alias": "..."}, ...],
  "max_recent": 10
}
```

**新增函数**：
- `load_project_presets()` / `save_project_presets()`
- `add_recent_project_path(filepath)`
- `get_recent_project_paths()`
- `add_project_bookmark(path, alias)`
- `remove_project_bookmark(path)`
- `get_project_bookmarks()`

### 2. project_bookmarks 模块

新建 `mmy_toolkit/project_bookmarks/` 目录：

| 文件 | 说明 |
|------|------|
| `__init__.py` | 模块入口、handlers 注册 |
| `properties.py` | 属性定义（书签、最近路径） |
| `operators.py` | 操作符（打开目录、添加书签等） |
| `ui.py` | 菜单组件 |

### 3. 自动记忆机制

使用 `bpy.app.handlers` 监听文件事件：

- `save_post` - 文件保存时记录路径
- `load_post` - 文件打开时记录路径

### 4. UI 挂载位置

| 位置 | 功能 |
|------|------|
| File Browser Header | 「打开目录」按钮（后缀按钮旁） |
| N 面板 - MMY工具 | 项目快捷访问区块（顶部） |

## 操作符列表

| bl_idname | 功能 |
|-----------|------|
| `mmy.open_project_folder` | 打开项目文件夹 |
| `mmy.add_project_bookmark` | 添加当前路径到书签 |
| `mmy.remove_project_bookmark` | 移除书签 |
| `mmy.open_bookmark_file` | 打开书签中的文件 |
| `mmy.open_bookmark_folder` | 打开书签文件所在文件夹 |
| `mmy.copy_project_path` | 复制项目路径到剪贴板 |
| `mmy.clear_recent_project_paths` | 清空历史 |
