# 备份容量能量条功能方案

## 功能概述

在 Blender 底部状态栏显示备份容量的"能量条"效果：
- 一格一格逐渐填满，直观展示容量使用比例
- 颜色从绿色→黄色→橙色→红色变化，提醒用户清理

## UI 效果示意

```
状态栏: [MMY] ████████░░ 320MB | 下次: 14:32
         ↑ 绿色填充，黄色/橙色/红色随容量变化
```

**颜色变化规则**：
| 容量比例 | 颜色 | 状态 |
|----------|------|------|
| 0-50% | 绿色 | 正常 |
| 50-75% | 黄色 | 注意 |
| 75-90% | 橙色 | 警告 |
| 90-100%+ | 红色 | 需清理 |

## 技术方案

### 方案选择

Blender UI 系统不支持直接设置 label 颜色，有以下方案：

| 方案 | 优点 | 缺点 |
|------|------|------|
| **A. Unicode 方块字符** | 简单，无需 GPU | 颜色固定，需用不同字符模拟 |
| **B. GPU 自定义绘制** | 真正的颜色变化 | 复杂，需管理绘制回调 |
| **C. 图标 + 文字组合** | 简单，有颜色提示 | 格子效果不明显 |

**推荐方案 A**：使用 Unicode 方块字符 `█` `░` 模拟格子效果，配合图标颜色提示。

### 实现细节

```python
def draw_statusbar_backup(self, context):
    layout = self.layout
    row = layout.row(align=True)

    # 计算填充比例
    capacity = status['capacity_mb']
    threshold = prefs.capacity_threshold_mb
    ratio = min(capacity / threshold, 1.0)

    # 生成能量条字符串（10格）
    filled = int(ratio * 10)
    bar = "█" * filled + "░" * (10 - filled)

    # 根据比例选择图标颜色提示
    if ratio < 0.5:
        icon = 'CHECKMARK'  # 绿色提示
    elif ratio < 0.75:
        icon = 'ERROR'      # 黄色提示
    elif ratio < 0.9:
        icon = 'CANCEL'     # 橙色提示
    else:
        icon = 'X'          # 红色提示
        row.alert = True    # 整行红色警告

    row.label(text=f"[MMY] {bar} {capacity:.0f}MB", icon=icon)
```
