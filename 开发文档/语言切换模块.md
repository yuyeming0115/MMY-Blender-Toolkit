# 语言切换模块

## 功能概述

一键切换 Blender 界面中/英文，从 Quick Translation 插件整合而来。

---

## 核心功能

| 功能 | 操作 |
|------|------|
| 切换语言 | 点击按钮切换中/英文，按钮显示当前状态："中"或"En" |
| 打开设置 | Ctrl + 点击打开偏好设置 |
| 显示位置 | 可配置：顶栏、Shader Editor、属性面板、3D视图 |

---

## 文件结构

```
mmy_toolkit/translation_toggle/
├── __init__.py      # 模块入口
├── operators.py     # MMY_OT_ToggleTranslation 操作符
└── ui.py            # Header 按钮绘制 + 位置配置
```

---

## 偏好设置属性

| 属性 | 说明 | 默认值 |
|------|------|--------|
| `translation_topbar` | 顶栏菜单显示 | True |
| `translation_node_header` | Shader Editor 显示 | True |
| `translation_properties_header` | 属性面板显示 | True |
| `translation_view3d_header` | 3D视图显示 | False |

---

## 关键 API

```python
# 切换翻译
view = context.preferences.view
view.use_translate_interface = not view.use_translate_interface
view.use_translate_tooltips = view.use_translate_interface
# 不切换 use_translate_new_dataname，保持用户原有设置
```

---

## 与原插件差异

| 原插件 | MMY 整合版 |
|--------|-----------|
| 自定义图标 | 显示文字"中"/"En"，按钮宽度紧凑（ui_units_x=1.0） |
| 独立偏好设置页 | 整合到 MMY 偏好设置 |
| Shift 反转图标颜色 | 移除（简化） |
| 翻译字典注册 | 移除（Blender 已有中文翻译） |
