# 智能选择模块开发文档

## 模块概述

智能选择模块提供双击智能选择功能，根据上下文自动判断选择类型。

## 功能列表

| 功能 | 场景 | 快捷键 | 说明 |
|------|------|--------|------|
| 选中 UV 孤岛 | UV 编辑器 | Shift+双击 | 选中整个相连的 UV 区域 |
| 选中缝合边 | 3D 视图（编辑模式） | 菜单 | 选中所有 UV 缝合边 |
| 选中相同材质 | 3D 视图（编辑模式） | Shift+双击 | 选中所有相同材质的面 |

## 文件结构

```
smart_select/
├── __init__.py    # 模块入口
├── operators.py   # 操作符定义
├── utils.py       # 选择逻辑工具函数
├── ui.py          # 偏好设置 UI
```

## 偏好设置属性

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `smart_select_enabled` | Bool | True | 启用智能选择功能 |
| `smart_select_double_click_interval` | Float | 0.25 | 双击检测间隔（秒） |
| `smart_select_mode` | Enum | 'auto' | 选择模式（auto/island/material/seam） |

## API 参考

### operators.py

```python
# 操作符列表
MMY_OT_SmartSelectUVIsland    # 选中 UV 孤岛
MMY_OT_SmartSelectUVSeams     # 选中缝合边
MMY_OT_SmartSelectMaterial    # 选中相同材质
MMY_OT_SmartSelectListener    # 双击监听器
```

### utils.py

```python
# 工具函数
select_uv_island(context, mouse_x, mouse_y)  # 选中 UV 孤岛
select_uv_seams(context)                      # 选中缝合边
select_faces_by_material(context, mouse_x, mouse_y)  # 选中相同材质
get_context_type(context)                     # 获取上下文类型
```
