# 智能选择模块开发文档

## 模块概述

智能选择模块提供双击智能选择功能，使用 Blender 原生 keymap 双击事件实现。

## 功能列表

| 功能 | 场景 | 快捷键 | 说明 |
|------|------|--------|------|
| 选中 UV 孤岛 | UV 编辑器 | 双击 | 选中整个相连的 UV 区域 |
| 选中相连元素 | 3D 视图（编辑模式） | 双击 | 根据选择模式选中相连点/边/面 |
| 选中相同材质 | 3D 视图（编辑模式） | Shift+双击 | 选中所有相同材质的面 |
| 选中缝合边 | 3D 视图（编辑模式） | Ctrl+双击 | 选中所有 UV 缝合边 |

## 文件结构

```
smart_select/
```

## 实现方式

### 关键技术点

使用 **Blender 原生 keymap 双击事件** 而非手动检测：
- `type='LEFTMOUSE', value='DOUBLE_CLICK'` 直接绑定到双击事件
- 操作符调用原生 API：`bpy.ops.uv.select_linked_pick('INVOKE_DEFAULT')`

### Keymap 注册

```python
# UV Editor - 双击选中孤岛
km.keymap_items.new(
    "mmy.smart_select_uv_island",
    type='LEFTMOUSE',
    value='DOUBLE_CLICK'
)

# Mesh - Shift+双击选中相同材质
km.keymap_items.new(
    "mmy.smart_select_material",
    type='LEFTMOUSE',
    value='DOUBLE_CLICK',
    shift=True
)
```

## API 参考

### operators.py

```python
# 操作符列表

MMY_OT_SmartSelectMaterial    # Shift+双击选中相同材质
```
