# 智能命名模块

## 功能概述

解决 Blender 原生复制自动添加 `.001` 后缀的问题，提供智能复制和批量重命名功能。

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 智能复制集合 | 复制集合内容，名称数字递增（如 `集合1` → `集合2`） |
| 智能复制对象 | 复制对象，名称数字递增 |
| 批量重命名 | 前缀 + 后缀 + 序号递增 |
| 预设系统 | 常用前缀/后缀预设存储 |

## 操作符列表

| bl_idname | 功能 |
|-----------|------|
| `mmy.smart_duplicate_collection` | 智能复制集合 |
| `mmy.smart_duplicate_object` | 智能复制对象 |
| `mmy.batch_rename` | 批量重命名（弹出面板） |
| `mmy.rename_single` | 单个对象智能重命名 |
| `mmy.apply_prefix` | 应用前缀预设 |
| `mmy.apply_suffix` | 应用后缀预设 |

---

## 预设系统

**存储文件**：`presets/naming_presets.json`

**默认预设**：
```json
{
  "separator": "_",
  "digits": 2,
  "prefix_presets": ["Model", "Char", "Prop", "Set", "Asset"],
  "suffix_presets": ["_01", "_LOD", "_A", "_B", "_High", "_Low"]
}
```

---

## 命名逻辑

```python
# 数字递增
smart_increment_name("Char_01", '_', 2) → "Char_02"
smart_increment_name("集合1", '', 1) → "集合2"
smart_increment_name("Model", '_', 2) → "Model_01"

# 批量重命名
apply_prefix_suffix("Body", "Char", "", 1, '_', 2) → "CharBody_01"
```
