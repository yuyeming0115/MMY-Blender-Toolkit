# 清理工具模块开发记录

## 功能概述

将独立插件 `MMY_Cleaning&Replace` 的清理功能整合到 MMY Toolkit，精简为 3D 视窗 Header 上的一个按钮，点击后弹出菜单提供各项清理操作。

## 二、清理功能列表

| 操作符 ID | 功能 | 说明 |
|-----------|------|------|
| `mmy.clean_all` | 一键清理 | 执行所有清理操作 |
| `mmy.clean_missing_images` | 删除丢失图片 | 文件路径已不存在的图片数据块 |
| `mmy.clean_unused_materials` | 删除未用材质 | 未被物体使用的材质球 |
| `mmy.clean_unused_images` | 删除未用贴图 | 未被引用的贴图数据 |
| `mmy.clean_invalid_references` | 删除无效引用 | 空集合 + 无引用节点组 |
| `mmy.clean_unused_animations` | 清理动画 | 空/未用Action、空NLA轨道 |
| `mmy.merge_duplicate_materials` | 合并重复材质 | .001 等重复材质和贴图 |
| `mmy.repair_purple_materials` | 一键同名替换 | 修复FBX导入的紫色丢失材质 |

## 四、关键技术

### 紫色材质判断

FBX 导入丢失贴图后，材质会显示为紫色。判断条件：
- 使用 Principled BSDF 着色器
- Base Color 为默认紫色 `#BF26BF` `(0.749, 0.149, 0.749, 1.0)`

```python
PURPLE_COLOR = (0.7490196, 0.1490196, 0.7490196, 1.0)

def is_purple_material(mat) -> bool:
    if not mat or not mat.use_nodes:
        return False
    for node in mat.node_tree.nodes:
        if node.type == 'BSDF_PRINCIPLED':
            base_color = node.inputs["Base Color"].default_value
            if abs(base_color[0] - PURPLE_COLOR[0]) < 0.01 and \
               abs(base_color[1] - PURPLE_COLOR[1]) < 0.01 and \
               abs(base_color[2] - PURPLE_COLOR[2]) < 0.01:
                return True
    return False
```

### 清理名称后缀

Blender 复制/导入时会自动追加数字后缀（如 `.001`, `.002`）：

```python
def clean_name(name: str) -> str:
    return re.sub(r'\.\d{3}$', '', name)
```

### 材质替换

将旧材质的所有引用替换为新材质：

```python
def reassign_material(old_mat, new_mat):
    for obj in bpy.data.objects:
        for i, slot in enumerate(obj.material_slots):
            if slot.material == old_mat:
                obj.material_slots[i].material = new_mat
    if old_mat.users == 0:
        bpy.data.materials.remove(old_mat)
```

### Header 菜单挂载

```python
class MMY_MT_CleaningMenu(bpy.types.Menu):
    bl_idname = "MMY_MT_cleaning_menu"
    bl_label = "清理工具"
    
    def draw(self, context):
        layout = self.layout
        layout.operator("mmy.clean_all", text="⚡ 一键清理（全部）", icon='TRASH')
        layout.separator()
        # ... 其他操作符

def _draw_cleaning_button(self, context):
    self.layout.menu("MMY_MT_cleaning_menu", text="", icon='BRUSH_DATA')

# 注册时挂载
bpy.types.VIEW3D_HT_header.prepend(_draw_cleaning_button)
```
