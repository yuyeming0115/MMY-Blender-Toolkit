# 修改器工具模块

## 功能概述

一键控制所有修改器的视口显示状态，提供修改器批量操作功能。

---

## 核心功能

| 按钮 | 功能 | 说明 |
|------|------|------|
| 添加 | 多列菜单 | 4列布局，每类修改器带正确图标 |
| 显隐 | 智能切换 | 隐藏时记住每个修改器状态，恢复时还原 |
| 应用 | 保留形态键 | 调用 shapekey_tools 的保留形态键应用逻辑 |
| 删除 | 批量删除 | 一键删除所有修改器 |
| 展开/折叠 | 面板控制 | 全部展开或折叠修改器面板 |

---

## 显隐开关逻辑

1. **隐藏**：隐藏所有修改器，记住每个修改器原本的显隐状态
2. **恢复**：恢复之前记住的显隐状态（不是简单的全部显示）

状态存储：`obj["mmy_modifier_visibility"]`

---

## 操作符

| 操作符 | 功能 |
|------|------|
| `mmy.hide_all_modifiers` | 隐藏所有修改器并保存状态 |
| `mmy.restore_modifier_visibility` | 恢复修改器显隐状态 |
| `mmy.delete_all_modifiers` | 删除所有修改器 |
| `mmy.expand_all_modifiers` | 展开所有修改器面板 |
| `mmy.collapse_all_modifiers` | 折叠所有修改器面板 |
| `mmy.apply_all_modifiers_with_shapekeys` | 保留形态键应用所有修改器 |

## 图标名称映射

### 问题解决

自定义多列菜单最初报错 `enum 'MOD_DECIMATE' not found`，原因是图标名称错误：
- ❌ `MOD_DECIMATE`
- ✅ `MOD_DECIM`

### Blender 5.1 可用图标（63 个）

获取方法：`bpy.types.UILayout.bl_rna.functions["prop"].parameters["icon"].enum_items`

主要图标：
```
MOD_ARRAY, MOD_BEVEL, MOD_BOOLEAN, MOD_BUILD, MOD_DECIM,
MOD_EDGESPLIT, MOD_MASK, MOD_MIRROR, MOD_MULTIRES, MOD_REMESH,
MOD_SCREW, MOD_SKIN, MOD_SOLIDIFY, MOD_SUBSURF, MOD_TRIANGULATE,
MOD_ARMATURE, MOD_CAST, MOD_CURVE, MOD_DISPLACE, MOD_LATTICE,
MOD_SHRINKWRAP, MOD_SIMPLEDEFORM, MOD_SMOOTH, MOD_CLOTH, MOD_OCEAN,
MOD_PARTICLES, MOD_SOFT...
```

### 修改器→图标映射表

见 `ui.py` 中的 `MODIFIER_ICON_NAMES` 字典。
