# 修改器工具模块

## 功能概述

一键控制所有修改器的视口显示状态（show_viewport）。

---

## 核心功能

| 功能 | 操作 |
|------|------|
| 切换显示 | 点击按钮一键切换所有修改器的视口显示状态 |
| 智能切换 | 全部显示时点击隐藏，有隐藏时点击显示 |
| 显示位置 | 可配置：数据属性面板、属性面板 Header |

---

## 文件结构

```
mmy_toolkit/modifier_tools/
├── __init__.py      # 模块入口
├── operators.py     # MMY_OT_ToggleAllModifiersViewport 操作符
└── ui.py            # Header 按钮绘制 + 位置配置
```

---

## 偏好设置属性

| 属性 | 说明 | 默认值 |
|------|------|--------|
| `modifier_data_header` | 数据属性面板 Header 显示 | True |
| `modifier_properties_header` | 属性面板 Header 显示 | False |

---

## 关键 API

```python
# 获取修改器列表
obj = context.active_object
modifiers = obj.modifiers

# 切换视口显示
for mod in modifiers:
    mod.show_viewport = new_state
```

---

## 按钮状态逻辑

- 全部显示 (`all_visible = True`) → 显示图标 `HIDE_OFF`，文字"全部隐藏"
- 有隐藏 (`all_visible = False`) → 显示图标 `HIDE_ON`，文字"全部显示"
