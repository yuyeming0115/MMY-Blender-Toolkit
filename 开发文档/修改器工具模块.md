# 修改器工具模块

## 功能概述

一键控制所有修改器的视口显示状态（show_viewport），并提供应用修改器功能。

---

## 核心功能

| 按钮 | 功能 | 说明 |
|------|------|------|
| 添加修改器 | 原生添加菜单 | 点击展开修改器类型列表 |
| 显隐开关 | 智能显隐切换 | 隐藏时记住每个修改器状态，恢复时还原 |
| 应用 | 保留形态键应用 | 调用 shapekey_tools 的保留形态键应用逻辑 |

---

## 显隐开关逻辑

1. **隐藏**：点击眼睛图标，隐藏所有修改器，并记住每个修改器原本的显隐状态
2. **恢复**：再次点击，恢复之前记住的显隐状态（不是简单的全部显示）

状态存储使用对象自定义属性 `obj["mmy_modifier_visibility"]`。

## 偏好设置属性

| 属性 | 说明 | 默认值 |
|------|------|--------|
| `modifier_panel` | 修改器面板显示 | True |
| `modifier_properties_header` | 属性面板 Header 显示 | False |

---

## 操作符

| 操作符 | 功能 |
|------|------|
| `mmy.hide_all_modifiers` | 隐藏所有修改器并保存状态 |
| `mmy.restore_modifier_visibility` | 恢复修改器显隐状态 |
| `mmy.toggle_all_modifiers_viewport` | 简单切换全部显隐 |
