# 项目文件快速切换功能方案

## 功能概述

在3D视图Header添加按钮，点击显示当前工程文件所在目录的其他.blend文件列表，实现快速切换打开。

---

## UI效果示意

```
Header区域: ... | [📂 项目文件 ▼] | [50mm ▼] | ...

             ↓ 点击展开菜单
         ┌─────────────────────┐
         │ 📄 Model.blend   ✓  │ ← 当前文件标记
         │ 📄 Model_Mat.blend  │
         │ 📄 Model_Rig.blend  │
         ├─────────────────────┤
         │ 📁 打开目录...      │
         └─────────────────────┘
```

---

## 技术方案

### 模块结构

```
mmy_toolkit/project_switcher/
├── __init__.py          # 模块入口
└── operators.py         # 操作符 + Menu + Header绘制
```

### 核心组件

| 组件 | 说明 |
|------|------|
| `MMY_MT_ProjectFiles` | 下拉菜单，动态扫描目录.blend文件 |
| `MMY_OT_OpenProjectFile` | 打开指定文件 |
| `MMY_OT_OpenProjectDirectory` | 打开系统文件管理器 |
| `_draw_project_switcher` | Header按钮绘制函数 |

### 文件数量限制

- 最多显示10个文件
- 按修改时间排序（最近优先）

- 当前文件标记 ✓
- 打开目录按钮
