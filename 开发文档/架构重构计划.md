# MMY Blender Toolkit 架构重构计划

## 背景

现有 MMY Blender Toolkit 已有多个功能模块，但 UI 采用单面板设计，随着功能增加需要更好的模块化管理。计划重构为页签式 UI 架构。

---

## 当前架构分析

### 现有结构
```
mmy_toolkit/
├── __init__.py              # 主入口（注册/注销 + 后缀保存功能）
├── config.py                # 配置存储
├── utils.py                 # 公共工具
├── mesh_tools/              # 网格工具
├── asset_browser/           # 资产创建
├── camera_tools/            # 相机焦距
├── rig_tools/               # 骨骼（预留）
├── auto_color_space.py      # 材质颜色空间
├── save_helper/             # 文件保存（内嵌在主入口）
└── ui/                      # N面板（单面板 VIEW3D_PT_MMYMeshTools）
```

### 当前UI方式
- **单面板设计**：`VIEW3D_PT_MMYMeshTools` 在N面板显示
- **模式切换**：根据 `OBJECT/EDIT` 模式动态显示不同功能
- **bl_category**：使用 `"MMY工具"` 作为侧边栏标签

---

## 目标架构：页签式 UI

### 设计意图
```
[模] [UV] [动] [烘] [灯] [渲]   ← 横向页签按钮
┌────────────────────────────┐
│  当前模块的功能面板        │
│                            │
└────────────────────────────┘
```

### 页签划分

| 页签 | 功能域 | 当前内容 | 待迁移内容 |
|------|--------|----------|------------|
| **模** | 建模 | 资产创建、材质颜色空间、相机焦距、FBX导入、缝合边（共享） | 旧建模插件功能 |
| **UV** | UV处理 | 缝合边（共享） | 旧UV工具插件功能 |
| **动** | 动画 | 空（预留） | 旧动画插件功能 |
| **烘** | 烘焙 | 空（预留） | 旧烘焙插件功能 |
| **灯** | 灯光 | 空（预留） | 旧灯光插件功能 |
| **渲** | 渲染 | 空（预留） | 旧渲染插件功能 |

> **缝合边工具**：在「模」和「UV」两处都显示，方便不同工作流使用

### 不放入页签面板的功能

以下功能保持在原生位置，不在页签面板中重复：

| 功能 | 原生位置 | 理由 |
|------|----------|------|
| 后缀保存 | 文件浏览器 Header | 保存文件时直接点 |
| 材质颜色空间 | Shader Editor 右键/Header | 选节点时就在手边 |
| 相机焦距 | 3D 视图 Header（计划） | 调视角最方便 |

**原则**：原生位置优先，页签面板只放需要集中管理的工具。

## 实施计划

### Phase 1：UI架构重构（当前目标）

**修改文件**：
| 文件 | 改动 |
|------|------|
| `mmy_toolkit/__init__.py` | 添加 `mmy_active_tab` EnumProperty 注册 |
| `mmy_toolkit/ui/panels.py` | 重构为页签式面板，调用各模块绘制函数 |
| `mmy_toolkit/mesh_tools/__init__.py` | 添加 `draw_panel()` 函数（缝合边） |
| `mmy_toolkit/asset_browser/__init__.py` | 添加 `draw_panel()` 函数（资产创建） |
| `mmy_toolkit/camera_tools/__init__.py` | 添加 `draw_panel()` 函数（焦距） |
| `mmy_toolkit/auto_color_space.py` | 添加 `draw_panel()` 函数（材质切换） |

**核心改动细节**：

#### 1. 页签 EnumProperty 定义
```python
# mmy_toolkit/__init__.py 或新建 mmy_toolkit/ui/tabs.py

def get_tab_items():
    return [
        ('MODEL', "模", "建模工具"),
        ('UV', "UV", "UV工具"),
        ('ANIM', "动", "动画工具"),
        ('BAKE', "烘", "烘焙工具"),
        ('LIGHT', "灯", "灯光工具"),
        ('RENDER', "渲", "渲染工具"),
    ]

bpy.types.Scene.mmy_active_tab = bpy.props.EnumProperty(
    items=get_tab_items(),
    default='MODEL'
)
```

#### 2. 面板重构
```python
# mmy_toolkit/ui/panels.py

class VIEW3D_PT_MMYToolkit(bpy.types.Panel):
    bl_label = "MMY工具"
    bl_idname = "VIEW3D_PT_MMY_toolkit"
    bl_space_type = 'VIEW_3D'
    bl_region_type = 'UI'
    bl_category = "MMY工具"

    def draw(self, context):
        layout = self.layout

        # 页签行
        row = layout.row(align=True)
        row.prop(context.scene, "mmy_active_tab", expand=True)

        layout.separator()

        # 根据页签绘制内容
        tab = context.scene.mmy_active_tab

        if tab == 'MODEL':
            self._draw_model_tab(layout, context)
        elif tab == 'UV':
            self._draw_uv_tab(layout, context)
        elif tab == 'ANIM':
            self._draw_anim_tab(layout, context)
        elif tab == 'BAKE':
            self._draw_bake_tab(layout, context)
        elif tab == 'LIGHT':
            self._draw_light_tab(layout, context)
        elif tab == 'RENDER':
            self._draw_render_tab(layout, context)

    def _draw_model_tab(self, layout, context):
        """建模页签"""
        from ..mesh_tools import draw_panel
        from ..asset_browser import draw_panel as draw_asset
        from ..camera_tools import draw_panel as draw_camera
        from ..auto_color_space import draw_panel as draw_color

        # 缝合边工具（共享）
        draw_panel(layout, context)

        # 资产创建
        draw_asset(layout, context)

        # 材质颜色空间
        draw_color(layout, context)

        # 相机焦距
        draw_camera(layout, context)

    def _draw_uv_tab(self, layout, context):
        """UV页签"""
        from ..mesh_tools import draw_panel
        draw_panel(layout, context)  # 缝合边工具

        # 预留：未来UV工具

    def _draw_anim_tab(self, layout, context):
        layout.label(text="动画工具（待开发）")

    def _draw_bake_tab(self, layout, context):
        layout.label(text="烘焙工具（待开发）")

    def _draw_light_tab(self, layout, context):
        layout.label(text="灯光工具（待开发）")

    def _draw_render_tab(self, layout, context):
        layout.label(text="渲染工具（待开发）")
```

### Phase 2：逐步迁移旧插件功能

按优先级将旧插件功能移入对应页签：
1. 先迁移建模相关 → 「模」页签
2. 再迁移UV相关 → 「UV」页签
3. 依次迁移动画、烘焙、灯光、渲染

## 验证方式

1. 在 Blender 中安装插件，打开 N面板
2. 确认页签按钮横向显示
3. 点击各页签，确认内容切换正确
4. 验证现有功能（缝合边、资产创建、焦距）正常工作
5. 确认缝合边在「模」和「UV」两页签都显示

## 用户确认汇总

| 问题 | 决定 |
|------|------|
| UV 是否独立页签 | **保持独立「UV」** |
| 现有功能归类 | **归到「模」**（资产创建、材质颜色空间、相机焦距） |
| 缝合边工具位置 | **「模」和「UV」两处都显示** |
| 迁移策略 | **逐步迁移** |
