# 集合架构模板功能开发计划

## 功能概述

快速搭建项目集合架构，支持：
1. **预设模板系统**：保存常用架构为 JSON 配置，一键应用
2. **快速生成器**：弹窗 UI 自定义输入，灵活创建
3. **批量创建**：输入多个资产名，自动生成完整结构

---

## 交互设计

### 入口位置

| 入口 | 功能 |
|------|------|
| N面板「智能命名」→ 集合架构区块 | 主入口：模板选择、快速生成器、批量模式、模板管理 |
| 大纲右键（空白处） | 快捷入口：应用默认/最近模板初始化场景 |
| 大纲右键（集合上） | 扩展入口：将模板应用为当前集合的子架构 |

### 用户流程

**新建文件初始化**：
```
N面板 → 选择模板 → 点击"生成架构" → 自动创建到场景根集合
```

**批量创建资产结构**：
```
N面板 → 快速生成器 → 开启"批量模式" → 输入"Hero,Enemy,Boss" → 生成
结果：
  Assets/
    ├── Hero/_high, _low
    ├── Enemy/_high, _low
    ├── Boss/_high, _low
```

**大纲快捷初始化**：
```
大纲空白处右键 → "初始化集合架构" → 弹出模板选择菜单 → 点击应用
```

## 技术方案

### 数据存储

**文件路径**：`mmy_toolkit/presets/collection_templates.json`

**数据结构**：
```json
{
  "default_template": "烘焙工作流",
  "recent_template": "动画流程",
  "templates": {
    "烘焙工作流": {
      "root_name": "Assets",
      "children": ["Char", "Prop", "Set"],
      "auto_lod": true,
      "lod_suffixes": ["_high", "_low"]
    },
    "动画流程": {
      "root_name": "Project",
      "children": ["Mesh", "Rig", "Ani", "Render"],
      "auto_lod": false
    },
    "自定义": null  // 用户自定义模板
  }
}
```

### 模块文件

```
mmy_toolkit/smart_naming/
├── collection_templates.py    # 新增：模板管理逻辑
├── operators.py               # 扩展：新增生成操作符
├── ui.py                      # 扩展：N面板架构区块 + 大纲右键
```

### 操作符列表

| bl_idname | 功能 |
|-----------|------|
| `mmy.generate_collection_template` | 应用模板生成架构 |
| `mmy.quick_generate_collections` | 快速生成器（弹窗） |
| `mmy.batch_generate_collections` | 批量生成资产结构 |
| `mmy.save_collection_template` | 保存当前架构为模板 |
| `mmy.set_default_template` | 设置默认模板 |

## 开发步骤

1. **阶段一：核心功能**
   - 创建 `collection_templates.py` 模板管理
   - 实现 `generate_collection_template` 操作符
   - N面板添加「集合架构」区块

2. **阶段二：快速生成器**
   - 实现 `quick_generate_collections` 弹窗 UI
   - 支持批量模式

3. **阶段三：大纲快捷入口**
   - 大纲右键菜单添加快捷选项
   - 模板选择菜单（动态生成）

4. **阶段四：模板管理**
   - 模板编辑/删除 UI
   - 从现有场景提取架构保存为模板
