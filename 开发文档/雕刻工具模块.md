# 雅刻工具模块开发记录

## 功能概述

为雕刻模式右键菜单添加面组（Face Sets）快捷操作。

---

## 核心功能

| 功能 | 操作符 | 说明 |
|------|--------|------|
| 从遮罩创建 | `sculpt.face_sets_create` (mode='MASKED') | 基于遮罩创建新面组 |
| 从可见面创建 | `sculpt.face_sets_create` (mode='VISIBLE') | 基于可见面创建 |
| 全部 | `sculpt.face_sets_create` (mode='ALL') | 为所有面创建面组 |
| 切换显示 | `sculpt.face_set_change_visibility` | 切换当前面组可见性 |
| 反转可见性 | `sculpt.face_set_invert_visibility` | 反转所有面组可见性 |
| 隐藏全部 | `sculpt.face_set_hide_all` | 隐藏所有面组 |
| 显示全部 | `sculpt.face_set_show_all` | 显示所有面组 |
| 填充 | `sculpt.face_set_edit` (mode='GROW') | 扩展面组范围 |
| 收缩 | `sculpt.face_set_edit` (mode='SHRINK') | 收缩面组范围 |

## 开发问题与解决

### 问题 1：原生菜单类名错误

| 错误 | 正确 |
|------|------|
| `VIEW3D_MT_sculpt_context_menu` | `VIEW3D_PT_sculpt_context_menu` |

**原因**：Blender 5.1 中雕刻右键菜单是 **Panel** 类型（PT），不是 **Menu** 类型（MT）。

**验证方法**：通过 `--debug-all` 启动 Blender，观察日志：
```
call wm.call_panel {'name': 'VIEW3D_PT_sculpt_context_menu'}
```

### 问题 2：面组创建 mode 参数错误

| 错误 | 正确 |
|------|------|
| `mode = 'MASK'` | `mode = 'MASKED'` |

**可用值**：`MASKED`, `VISIBLE`, `ALL`, `SELECTION`

### 问题 3：Header 入口位置不便

用户反馈 Header 按钮在雕刻过程中距离太远，不方便使用。

**解决方案**：移除 Header 入口，只保留右键菜单入口。

## 技术方案

### 扩展 Panel 类菜单

```python
def _draw_sculpt_context_menu(self, context):
    """扩展雕刻模式右键菜单"""
    obj = context.active_object
    if not obj or obj.type != 'MESH' or obj.mode != 'SCULPT':
        return

    layout = self.layout
    layout.separator()
    layout.menu("MMY_MT_sculpt_face_sets", text="面组", icon='GROUP_VERTEX')


def register():
    bpy.types.VIEW3D_PT_sculpt_context_menu.append(_draw_sculpt_context_menu)


def unregister():
    bpy.types.VIEW3D_PT_sculpt_context_menu.remove(_draw_sculpt_context_menu)
```
