# 雕刻工具模块开发记录

## 功能概述

为雕刻模式右键菜单添加面组（Face Sets）快捷操作。

## 核心功能

| 功能 | 操作符 | 说明 |
|------|--------|------|
| 从遮罩创建 | `sculpt.face_sets_create` (mode='MASK') | 基于遮罩创建新面组 |
| 切换显示 | `sculpt.face_set_change_visibility` | 切换当前面组可见性 |
| 反转可见性 | `sculpt.face_set_invert_visibility` | 反转所有面组可见性 |
| 隐藏全部 | `sculpt.face_set_hide_all` | 隐藏所有面组 |
| 显示全部 | `sculpt.face_set_show_all` | 显示所有面组 |
| 填充 | `sculpt.face_set_edit` (mode='GROW') | 扩展面组范围 |
| 收缩 | `sculpt.face_set_edit` (mode='SHRINK') | 收缩面组范围 |
