# 渲染预览图功能开发计划

## Context

用户需要一个快速渲染预览图的功能，用于生成资产的缩略图。当前工作流程中，用户需要手动渲染、手动裁剪、手动命名，效率较低。

**需求**：
- 一键渲染第 0001 帧
- 输出到工程文件所在的 `Render` 文件夹
- 文件名自动取当前 .blend 文件名去掉配置的后缀（如 `Model_Render.blend` → `Model.png`）
- 自动裁剪透明/空白边缘，只保留有内容的区域
- UI 挂载在 3D 视图 Header
- 渲染完成后自动打开 Render 文件夹
- 后缀去除列表可在偏好设置中配置，支持预设存储
- 下拉菜单提供临时选项：使用配置、不去除后缀、自定义后缀

## 实现方案

### 1. UI 挂载

**位置**：3D 视图 Header（`VIEW3D_HT_header`）

**方式**：通过 `append` 钩子添加按钮 + 下拉菜单

```python
def _append_render_button(self, context):
    layout = self.layout
    if context.mode == 'OBJECT':
        layout.separator()
        row = layout.row(align=True)
        row.operator("mmy.render_preview", text="", icon='RENDER_STILL')
        row.menu("MMY_MT_render_preview_menu", text="", icon='DOWNARROW_HLT')
```

### 2. 后缀预设系统

**存储位置**：偏好设置中的 `render_remove_suffixes` CollectionProperty

**默认后缀列表**：`_Render`, `_Preview`, `_Test`

**配置界面**：在插件偏好设置中显示后缀列表，支持添加/删除

```python
# 偏好设置中
render_remove_suffixes: bpy.props.CollectionProperty(type=MMY_SuffixItem)

# 初始化默认列表
default_render_suffixes = ["_Render", "_Preview", "_Test"]
for suffix in default_render_suffixes:
    item = prefs.render_remove_suffixes.add()
    item.name = suffix
```

### 3. 下拉菜单选项

| 选项 | 功能 |
|------|------|
| 使用配置 | 使用偏好设置中的后缀去除列表（默认） |
| 不去除后缀 | 保留原始文件名，不去除任何后缀 |
| 自定义后缀... | 弹出对话框，指定要去除的后缀 |

### 4. Operator 实现

**Operator ID**：`mmy.render_preview`

**核心流程**：
1. 获取当前 .blend 文件路径和文件名
2. 根据 `temp_suffix_mode` 处理后缀：
   - `default`：使用偏好设置中的后缀列表，按顺序匹配
   - `none`：不去除任何后缀
   - `custom`：去除指定的自定义后缀
3. 构建输出路径：`工程目录/Render/{name}.png`
4. 设置渲染参数（帧号 1、输出路径）
5. 执行渲染
6. 渲染完成后裁剪图片
7. 自动打开 Render 文件夹

### 5. 图片裁剪方案

**方案选择**：使用 Python + Pillow 库进行裁剪（优先）

**备选方案**：如果 Pillow 不可用，使用 Blender 内置的图像处理 API

### 6. Blender 5.x API 适配

**注意**：Blender 5.x 中 `file_format` 属性移到了 `image_settings` 下

```python
# Blender 5.x 正确写法
scene.render.image_settings.file_format = 'PNG'
scene.render.image_settings.color_mode = 'RGBA'
```

2. 点击 Header 渲染按钮（默认使用配置）
3. 检查 `Render/Test.png` 是否生成
4. 使用下拉菜单选择"不去除后缀"
5. 检查 `Render/Test_Render.png` 是否生成
6. 在偏好设置中添加/删除后缀
7. 验证后缀列表是否持久保存

## 边界情况处理

- 文件未保存：提示用户先保存文件
- 文件名不匹配任何后缀：直接使用原文件名
- Render 目录不存在：自动创建
- 渲染区域全透明：不裁剪，保留原图
- Pillow 未安装：使用 Blender 内置 API 处理

## 技术问题记录

### 问题一：Blender 5.x file_format API 变化
**现象**：`AttributeError: 'RenderSettings' object has no attribute 'file_format'`

**原因**：Blender 5.x 将 `file_format` 移到了 `image_settings` 下

**解决**：使用 `scene.render.image_settings.file_format`
