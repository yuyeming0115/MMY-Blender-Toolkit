# 渲染预览图功能开发计划

## Context

用户需要一个快速渲染预览图的功能，用于生成资产的缩略图。当前工作流程中，用户需要手动渲染、手动裁剪、手动命名，效率较低。

**需求**：
- 一键渲染第 0001 帧
- 输出到工程文件所在的 `Render` 文件夹
- 文件名自动取当前 .blend 文件名去掉 `_Render` 后缀（如 `Model_Render.blend` → `Model.png`）
- 自动裁剪透明/空白边缘，只保留有内容的区域
- UI 挂载在 3D 视图 Header
- 渲染完成后自动打开 Render 文件夹

## 实现方案

### 1. UI 挂载

**位置**：3D 视图 Header（`VIEW3D_HT_header`）

**方式**：通过 `append` 钩子添加按钮

```python
def _append_render_button(self, context):
    layout = self.layout
    layout.operator("mmy.render_preview", text="", icon='RENDER_STILL')

bpy.types.VIEW3D_HT_header.append(_append_render_button)
```

### 2. Operator 实现

**Operator ID**：`mmy.render_preview`

**核心流程**：
1. 获取当前 .blend 文件路径和文件名
2. 解析文件名，去掉 `_Render` 后缀
3. 构建输出路径：`工程目录/Render/{name}.png`
4. 设置渲染参数（帧号 1、输出路径）
5. 执行渲染
6. 渲染完成后裁剪图片
7. 自动打开 Render 文件夹

### 3. 图片裁剪方案

**方案选择**：使用 Python + Pillow 库进行裁剪

**裁剪逻辑**：
- 加载渲染的 PNG 图片
- 分析像素数据，找到非透明区域的边界
- 裁剪并保存

**备选方案**：如果 Pillow 不可用，使用 Blender 内置的图像处理 API

```python
# 裁剪核心代码（Pillow 方案）
from PIL import Image

def crop_transparent_border(image_path):
    img = Image.open(image_path)
    # 获取非透明区域的边界框
    bbox = img.getbbox()
    if bbox:
        cropped = img.crop(bbox)
        cropped.save(image_path)
```

### 4. 文件路径处理

```python
def get_output_path():
    blend_path = bpy.data.filepath
    blend_dir = os.path.dirname(blend_path)
    blend_name = os.path.splitext(os.path.basename(blend_path))[0]

    # 去掉 _Render 后缀
    if blend_name.endswith("_Render"):
        output_name = blend_name[:-7]  # 去掉 "_Render"
    else:
        output_name = blend_name

    # 构建 Render 目录路径
    render_dir = os.path.join(blend_dir, "Render")
    if not os.path.exists(render_dir):
        os.makedirs(render_dir)

    output_path = os.path.join(render_dir, f"{output_name}.png")
    return output_path
```

### 5. 渲染参数设置

```python
def setup_render_settings(output_path):
    scene = bpy.context.scene

    # 设置帧号为 1
    scene.frame_set(1)

    # 设置输出路径
    scene.render.filepath = output_path
    scene.render.file_format = 'PNG'
    scene.render.image_settings.color_mode = 'RGBA'  # 保留透明通道用于裁剪
```

### 6. 渲染执行方式

**方案 A**：直接渲染（阻塞式）
```python
bpy.ops.render.render(write_still=True)
# 渲染完成后裁剪
crop_transparent_border(output_path)
```

**推荐**：方案 A，简单可靠，预览图渲染通常很快

### 7. 打开文件夹

```python
import subprocess
import sys

def open_folder(path):
    if sys.platform == 'win32':
        subprocess.run(['explorer', path])
    elif sys.platform == 'darwin':
        subprocess.run(['open', path])
    else:
        subprocess.run(['xdg-open', path])
```

## 边界情况处理

- 文件未保存：提示用户先保存文件
- 文件名不含 `_Render`：直接使用原文件名
- Render 目录不存在：自动创建
- 渲染区域全透明：不裁剪，保留原图
- Pillow 未安装：使用 Blender 内置 API 处理
